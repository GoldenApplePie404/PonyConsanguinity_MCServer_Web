# 生存服游玩指南

## 1. 加载资源包

服务器提供了美化游戏界面的专属资源包，玩家需要加载该资源包才能获得完整的游戏体验。

进入生存服后，系统会自动下载并加载该资源包，玩家无需手动操作。

### 手动添加方法
若自动加载失败，可尝试手动添加：
1. 从 [QQ 群](https://qm.qq.com/q/LHzmcKhmo0)文件中下载 [资源包](https://www.123865.com/s/nUR0Vv-hV4k3)
2. 将下载的资源包拖入当前游戏版本的 `resourcepacks` 文件夹
3. 进入游戏后，依次点击「选项」→「资源包」→「添加资源包」
4. 选择下载的资源包完成添加

### 资源包内容
- 粘液科技专属材质
- 特殊武器材质
- 特殊音效

## 2. 熟悉游戏菜单

服务器提供了便捷的操作菜单，集成了玩家常用的各类功能，如传送、邮箱、商店、领地、签到、称号等。菜单以钟为材质，右键即可打开。

### 菜单功能
- **个人信息**：查询等级、经济、ID 等信息
- **传送系统**：主城、家园、系统商店等传送功能
- **粘液科技**：提供配方查询和进度跟踪
- **礼包邮箱**：领取服务器发放的各类礼包
- **商店系统**：
  - 兑换商店：使用黄金券兑换物品
  - 系统商店：使用梦幻币购买物品
  - 全球商店：包含玩家上架的物品
  - 赞助商店：使用黄金券购买 VIP 月卡/年卡
- **举报功能**：提交举报信息，需填写被举报玩家 ID 和具体原因
- **技能系统**：查看个人技能等级
- **意见反馈**：提交游戏建议和问题
- **点歌系统**：请求播放音乐（暂未开放）
- **粒子特效**：编辑和使用个性化粒子效果（VIP 专属）
- **垃圾桶**：存放不需要的物品，也可临时存储重要物品
- **称号系统**：查看和兑换个人称号
- **签到系统**：每日签到领取奖励
- **药水拓展**：通过「蒸馏」合成特殊药水，可应用于装备并获得不同加成
- **梦幻卡池**：使用梦幻结晶进行抽卡，达到保底次数可获得特殊道具
- **每日任务**：完成任务获取黄金券、任务币、梦幻币和经验值
- **领地系统**：提供领地指令指南和权限操作菜单（需在领地内打开）
- **基础指令**：返回死亡地点、传送等常用指令帮助
- **更多附魔**：提供额外附魔的图鉴指南

## 3. 经济道具系统

生存服包含以下经济道具系统：

- **黄金券**：服务器稀有货币，可在各服务器间通用。获取方式包括：限时梦幻币兑换、黄金券兑换券（每日抽奖概率获得）、每日任务、任务币兑换、服务器活动福利。

- **任务币**：通过完成每日任务获得，可在商店中兑换物品。

- **梦幻币**：生存服专用普通货币，用于系统商店和玩家个人商店的交易。

- **梦幻结晶**：用于梦幻卡池抽卡，需使用黄金券购买，不同卡池对应不同类型的结晶（价格各异）。

- **梦幻之钥**：用于梦幻宝箱抽奖，仅可通过黄金券购买。梦幻宝箱位于主城，可通过传送菜单前往。

## 4. 常用指令

| 指令 | 功能说明 | 限制 |
|------|---------|------|
| `/deathback` | 返回最后一次死亡地点 | 无 |
| `/tpa <玩家ID>` | 向目标玩家发送传送请求 | 需对方同意 |
| `/sit` | 坐下 | VIP 专属 |
| `/fly` | 开启飞行模式 | VIP 专属 |
| `/feed` | 恢复饱食度 | VIP 专属 |
| `/heal` | 恢复生命值 | VIP 专属 |
| `/workbench` | 打开工作台 | VIP 专属 |
| `/anvil` | 打开铁砧 | VIP 专属 |
| `/ca sell <价格> [数量]` | 将手持物品上架系统商店 | 无 |
| `/ca buy <报酬> [数量] [物品]` | 从系统商店收购物品 | 无 |
| `/marry` | 结婚系统 | 无 |
| `/money pay <玩家ID> <金额>` | 向其他玩家转账梦幻币 | 无 |
| `/fhb <……>` | 发送红包 | 无 |
| `/msg <玩家ID> <消息>` | 发送悄悄话 | 无 |
| `/afk` | 进入挂机模式 | 无 |

### 其他功能

- **箱子锁**：将牌子放在箱子上，使用 `/lock 行号 玩家名` 可设置可打开箱子的玩家

- **个人商店**：
  1. 放置一个箱子
  2. 手持商品，输入 `/qs create [价格]` 创建商店
  3. 出售商品时记得及时补货
  4. 购买商品：左键点击商店告示牌，在聊天栏输入购买数量
  5. 商家管理：右击商店告示牌打开管理界面

- **领地系统**：使用木棍作为工具，左键+右键圈定区域，输入 `/res create` 创建领地（需消耗梦幻币）

- **盔甲架编辑器**：使用燧石作为编辑工具，可免费编辑盔甲架，用于个性化建筑或展示建模创意

- 化学工程

详细教程可参考：[粘液科技 Wiki](https://slimefun-addons-wiki.guizhanss.cn/)

- 飞行权限：使用 `/fly` 开启飞行
- 自助餐：使用 `/feed` 恢复饱食度
- 治疗：使用 `/heal` 恢复生命值
- 便捷工具：使用 `/workbench` 和 `/anvil` 快速打开工作台和铁砧
- 死亡不掉落
- 签到额外奖励
- 更大的领地空间

- 更多专属特权
