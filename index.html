服务器简介

万驹同源（PonyConsanguinity）是一个完全公益的小马 Minecraft 服务器。我们致力于为广大玩家提供优质的游戏体验，拥有稳定的基础设施、友好的社区氛围和丰富的游戏玩法。无论你是新手还是老玩家，都能在这里找到属于自己的乐趣。

服务器类型：插件服，拥有丰富多样的特色功能和玩法。由于服务器使用了多版本兼容插件，可以使版本覆盖范围在 Java 1.12.x ~ 1.21.1（推荐使用 1.20.1 版本），让更多玩家能够畅玩。当前服务器仅对 Java 版开放，后续将考虑推出基岩版适配，敬请期待！

我们承诺提供公平公正的游戏环境，严禁任何作弊行为。所有玩家在规则面前人人平等，让我们一起创造一个和谐的游戏社区。万驹同源欢迎你的加入！
