#### 常用指令

**基础指令**：
- `/deathback` - 返回最后一次死亡地点
- `/tpa <玩家 ID>` - 向目标玩家发送传送请求
- `/sit` - 坐下（VIP 专属）
- `/afk` - 进入挂机模式
- `/msg <玩家 ID> <消息>` - 发送悄悄话

**VIP 专属指令**：
- `/fly` - 开启飞行模式
- `/feed` - 恢复饱食度
- `/heal` - 恢复生命值
- `/workbench` - 打开工作台
- `/anvil` - 打开铁砧

**经济指令**：
- `/ca sell <价格> [数量]` - 将手持物品上架系统商店
- `/ca buy <报酬> [数量] [物品]` - 从系统商店收购物品
- `/money pay <玩家 ID> <金额>` - 向其他玩家转账梦幻币
- `/fhb <……>` - 发送红包

**社交指令**：
- `/marry` - 结婚系统

#### 特色功能

- **箱子锁**：将牌子放在箱子上，使用 `/lock 行号 玩家名` 设置可打开箱子的玩家
- **个人商店**：放置箱子，手持商品，输入 `/qs create [价格]` 创建商店
- **领地系统**：使用木棍作为工具，左键 + 右键圈定区域，输入 `/res create` 创建领地
- **盔甲架编辑器**：使用燧石作为编辑工具，可免费编辑盔甲架
